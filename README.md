# GitHub Portfolio Generator

Transform any GitHub repository into a beautiful portfolio page. Perfect for showcasing your projects to both technical and non-technical audiences.

## Features

- **Beautiful Design** - Modern UI with glass-morphism effects and gradients
- **Project Stats** - Display stars, forks, and watchers
- **Technology Breakdown** - Visual representation of languages used
- **Project Screenshots** - Automatic image detection and display
- **Easy Sharing** - Share your portfolio via Twitter, LinkedIn, or Email
- **Live Demo Links** - Direct links to both GitHub and live demos

## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/yourusername/portfolio-generator.git
cd portfolio-generator
```

2. Install dependencies:
```bash
npm install
```

3. Create a `.env.local` file with your GitHub token:
```bash
GITHUB_TOKEN="your_token_here"
```

4. Run the development server:
```bash
npm run dev
```

5. Open [http://localhost:5173](http://localhost:5173) in your browser

## Usage

1. Enter any GitHub repository URL
2. Get an instant portfolio page
3. Share with anyone using the built-in share buttons

## Tech Stack

- React + TypeScript
- Vite
- TailwindCSS
- GitHub API (via Octokit)
